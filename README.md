# Mini_JS_Projects

Five small web applications built using **vanilla JavaScript, HTML, and CSS** 

## Projects

### 1. Ecommerce App
A simple shopping cart that lets users add/remove products, updates the total price dynamically, and persists cart data using **localStorage**.

### 2. To Do App
A dark-mode to-do list that allows users to add, complete, and delete tasks while saving them in **localStorage**.

### 3. Weather App
A weather application that fetches real-time weather data from the **OpenWeather API** and dynamically updates the UI.

### 4. Expense Tracker
A browser-based expense tracker that allows users to add and remove expenses while storing data using **localStorage**.

### 5. Quiz App
An interactive quiz application with **randomized question selection, dynamic DOM updates, and score tracking**.
